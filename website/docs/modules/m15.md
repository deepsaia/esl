# Module 15: Capstone

Put it all together. Pick a real problem, choose your evolutionary weapons, run rigorous experiments, and write it up. This is your portfolio piece.

## Project Ideas

1. **CMA-ES vs. the world**: Compare CMA-ES, NES, and a GA on a suite of BBOB functions with statistical rigor. Write a benchmarking report.
2. **Evolve a robot controller**: Use ES to train a neural network policy for a MuJoCo/Gymnasium locomotion task. Compare with PPO.
3. **Quality-Diversity robotics**: Use MAP-Elites to generate a diverse repertoire of robot gaits, then show damage adaptation.
4. **Multi-objective engineering design**: Solve a real engineering trade-off problem (e.g., truss design, portfolio optimization) with NSGA-II and visualize the Pareto front.
5. **Surrogate-assisted expensive optimization**: Apply surrogate-assisted CMA-ES to a computationally expensive simulation.
6. **Neuroevolution for game AI**: Evolve a NEAT agent to play a game, visualize the evolved topologies.

## Deliverables

- Working code repository with reproducible experiments
- Written report (5-10 pages) with problem description, method, results, and analysis
- Visualization of results (convergence plots, Pareto fronts, behavior archives, etc.)

## Evaluation Criteria

- Problem selection and motivation
- Correct implementation and experimental methodology
- Quality of analysis and visualization
- Code quality and reproducibility
