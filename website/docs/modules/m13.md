# Module 13: Benchmarking - BBOB, CEC, and Fair Comparisons

"My algorithm beats yours!" - famous last words before someone checks the experimental setup. Rigorous benchmarking is what separates science from folklore in optimization. This module teaches you to run experiments that actually mean something, using the standard benchmark suites the community trusts.

## Learning Objectives

1. Use the COCO/BBOB benchmark suite to evaluate ES algorithms.
2. Understand CEC competition benchmarks and their test function properties.
3. Implement proper experimental methodology: multiple runs, statistical tests, performance profiles.
4. Generate empirical cumulative distribution function (ECDF) plots and convergence graphs.
5. Avoid common benchmarking pitfalls (cherry-picking, unfair comparisons, overfitting to benchmarks).
