# Module 05: CMA-ES Advanced - Variants, Restarts, and Large-Scale

Vanilla CMA-ES is powerful, but real problems are messy. This module covers the battle-tested variants: IPOP/BIPOP restart strategies for multimodal landscapes, sep-CMA-ES for high dimensions, and the latest large-scale variants that break through the curse of dimensionality.

## Learning Objectives

1. Implement and compare IPOP-CMA-ES and BIPOP-CMA-ES restart strategies.
2. Understand sep-CMA-ES and VD-CMA for scaling to hundreds/thousands of dimensions.
3. Use CMA-ES with bounded constraints and injected solutions.
4. Tune CMA-ES hyperparameters (population size, initial sigma, initial mean) for practical problems.
5. Apply CMA-ES to a real-world optimization problem end-to-end.
