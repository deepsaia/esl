# Module 10: Multi-Objective Optimization - Pareto Fronts and Beyond

Real-world problems rarely have a single objective. You want a car that's fast AND fuel-efficient. A drug that's effective AND has few side effects. Multi-objective evolutionary algorithms find the entire trade-off surface (the Pareto front) in a single run - no manual weight-tuning required.

## Learning Objectives

1. Understand Pareto dominance, Pareto fronts, and non-dominated solutions.
2. Implement NSGA-II from scratch (fast non-dominated sorting + crowding distance).
3. Use `pymoo` to solve multi-objective benchmark problems (ZDT, DTLZ).
4. Understand MOEA/D (decomposition-based approach) and compare it to NSGA-II.
5. Visualize 2D and 3D Pareto fronts and compute hypervolume indicators.
