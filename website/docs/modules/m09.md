# Module 09: Genetic Algorithms - Selection, Crossover, and Mutation

Genetic algorithms are the rowdy older cousin of evolution strategies. Where ES focuses on continuous optimization with Gaussian mutations, GAs bring crossover, discrete representations, and a colorful zoo of selection methods. Understanding both families - and when to use which - makes you a complete evolutionary computation practitioner.

## Learning Objectives

1. Implement a GA from scratch: binary and real-valued representations.
2. Code tournament selection, roulette-wheel selection, and elitism.
3. Implement crossover operators (single-point, uniform, SBX) and mutation.
4. Solve the Traveling Salesman Problem (TSP) with a GA using order crossover.
5. Understand the GA vs. ES distinction: when crossover helps and when it hurts.
