# Module 07: Neuroevolution — Evolving Neural Networks

Forget backpropagation (just for this module). What if we evolved the weights — or even the architecture — of neural networks? From NEAT's elegant complexification to modern weight-space evolution, neuroevolution has produced surprisingly competitive results and some genuinely alien-looking network topologies.

## Learning Objectives

1. Implement weight evolution of a simple neural network using ES.
2. Understand NEAT (NeuroEvolution of Augmenting Topologies) and run it on a classic task.
3. Explore HyperNEAT and indirect encoding schemes.
4. Compare neuroevolution vs. gradient-based training on small supervised learning tasks.
5. Understand when neuroevolution is a practical choice.
