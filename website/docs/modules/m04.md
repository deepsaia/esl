# Module 04: CMA-ES — The Crown Jewel of Evolution Strategies

CMA-ES (Covariance Matrix Adaptation Evolution Strategy) is arguably the most successful derivative-free optimizer for continuous domains. It learns the shape of the fitness landscape by adapting a full covariance matrix — think of it as giving your population an evolving "intuition" about which directions are promising. This is the module you'll keep coming back to.

## Learning Objectives

1. Understand the geometric intuition behind covariance matrix adaptation (sampling from an evolving ellipsoid).
2. Walk through the CMA-ES algorithm step by step: mean update, rank-one update, rank-mu update, step-size control (CSA).
3. Use the `cma` library (pycma) to solve benchmark problems.
4. Implement a simplified CMA-ES from scratch in NumPy to demystify the math.
5. Understand when CMA-ES excels and when it struggles.
