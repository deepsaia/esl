# Module 01: Optimization Foundations — Why Derivative-Free Methods Matter

Before evolving anything, we need to understand what optimization is and why sometimes you cannot (or should not) compute gradients. From noisy loss landscapes to black-box simulators, this module reveals the vast world where gradient descent cannot tread.

## Learning Objectives

1. Distinguish gradient-based, gradient-free, and zeroth-order optimization methods.
2. Understand when derivative-free methods are necessary (noisy, discontinuous, black-box objectives).
3. Implement random search, grid search, and simple hill climbing from scratch.
4. Benchmark these naive methods on standard test functions (Sphere, Rastrigin, Rosenbrock).
5. Visualize fitness landscapes and understand local vs. global optima.
