# Module 00: Setup & Tools

Get your evolutionary lab up and running. We'll install the key libraries, run your first optimization, and watch a population of solutions swarm toward a minimum like ants finding sugar.

## Learning Objectives

1. Install Python and core ES libraries (`cma`, `nevergrad`, `deap`).
2. Run your first ES optimization on a simple 2D function.
3. Visualize population evolution over generations.
4. Understand the evaluate-select-mutate loop that underpins all evolutionary methods.
