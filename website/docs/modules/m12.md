# Module 12: Constrained & Combinatorial Optimization

The real world doesn't give you smooth, unconstrained functions to optimize. It gives you budgets, safety limits, integer variables, and feasibility regions shaped like gerrymandered districts. This module teaches you how to handle constraints, discrete variables, and combinatorial explosions with evolutionary methods.

## Learning Objectives

1. Implement constraint handling techniques: penalty methods, feasibility rules, epsilon-constraint.
2. Solve constrained continuous optimization problems with CMA-ES + penalty functions.
3. Adapt ES/GA for combinatorial problems (permutation-based representations).
4. Solve a real combinatorial problem (job-shop scheduling or knapsack) with evolutionary methods.
5. Understand mixed-integer optimization and how evolutionary methods handle hybrid search spaces.
