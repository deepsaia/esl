# Module 06: Natural Evolution Strategies (NES)

What if we treated evolution as a machine learning problem - optimizing the parameters of a probability distribution using gradients of expected fitness? NES bridges the gap between evolution strategies and policy gradients, and the math is surprisingly elegant. If you squint, REINFORCE and NES are the same algorithm wearing different hats.

## Learning Objectives

1. Derive the score function estimator (log-likelihood trick) for expected fitness.
2. Implement vanilla NES (REINFORCE-style gradient estimation) from scratch.
3. Understand fitness shaping (rank-based utilities) and why it stabilizes learning.
4. Implement xNES and compare it to CMA-ES on benchmark functions.
5. Connect NES to policy gradient methods in RL.
